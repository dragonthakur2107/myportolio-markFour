Making our first CLI APP

What is a CLI APP ?

The best answer to this is apps we use, for example when we want to post something on our instagram story, we would click on add story. The way we interact with the system is by clicking a button, this is known as GUI(Graphical user interface).

Similarly CLI means command line interface, where we right a code to interact with the system.

Requirements?

To get started you will need this things:-

  • A laptop or PC with Chrome installed.
  • Good internet connection.
  • Create an account on repl.it

Now we are ready to go.

Steps:-

How to open a new repl-

  • You will be on home page of repl.it, now click "+ New repl".
  • Now a small window will open. As we are going to code in javascript, type "node.js" , select it and create repl.

Now we will be able to write our program and see it run.

How to output (print) something?

To see how, we will output our name-

  • Type "console.log("Your_name") " as shown.
    • Red is for code, Green is for output.

    How to read inputs?

    • Add readlineSync() as a dependency
    • Now create a new variable (in my case userName) and store ask user to give his/her name.
